# 🔁 数据绑定

数据绑定提供了一种简单的方式，在不需要每次值发生变化时都设置控件的属性的情况下，将数据传递到应用程序的用户界面中。

数据绑定是一种建立对象属性和UI控件属性之间映射的方式。这些映射可以是双向的，即在控件中的更改会应用到基础对象中，反之亦然，当对象属性的更改导致UI更改时，或者是单向的，只有在一个方向上传递更改。例如，如果用户在TextBox中编辑值，则底层对象的属性将自动更新以包括用户输入的值。

绑定通常与[MVVM模式](https://msdn.microsoft.com/zh-cn/library/hh848246.aspx)一起使用，本指南的其余部分将假定您在代码中使用该模式。
